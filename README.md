# Corporate-Performance-Social-Media-Insights-Dashboard
Project Overview
This project explores the intersection of regional financial health and global social media engagement. Using a pre-cleaned dataset, I developed a high-level reporting tool to visualise how digital presence correlates with revenue and debt management across different territories.

My Contribution
For this project, I was responsible for the analytical logic and visual construction:

The Calculation Engine: I built the back-end logic to aggregate social metrics (TikTok, LinkedIn, YouTube, etc.) and financial data into summary tables.

Dashboard Development: I designed and built the interactive dashboard, including multi-year trend analysis and regional performance rankings.

Key Features
Regional Finance: Revenue vs Expenses tracking across US regions (The West, South, etc.).

Digital Footprint: Analysis of platform reach, page views, and visitor behaviour.

Interactivity: Filterable views by Country and Year to drill down into specific performance windows.
