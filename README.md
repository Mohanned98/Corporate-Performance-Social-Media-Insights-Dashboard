# Corporate-Performance-Social-Media-Insights-Dashboard
.
Project Overview
This project features a comprehensive, multi-layered dashboard developed in Excel, designed to provide a holistic view of a fictional company's financial health and digital presence. The dashboard transforms raw, disparate data into actionable insights, focusing on financial performance, regional workforce distribution, and social media engagement across global markets.

I was responsible for the end-to-end development of this project, including data cleaning, complex formula modelling (calculations), and the final UI/UX design of the dashboard.

Key Features
1. Financial Performance Analysis
Revenue & Expense Tracking: A multi-year comparison (2020–2023) of financial data across different US regions (The West, South, North-East, and Mid-West).

Profitability Insights: Visualisation of revenue vs. expenses to identify high-performing states and regions.

Debt Management: Monitoring of 'Late Debtors' to assess credit risk and financial stability.

2. Digital Marketing & Social Media Metrics
Platform Reach: Comparative analysis of engagement across Facebook, Twitter, TikTok, LinkedIn, and YouTube.

Web Analytics: Tracking of page views, average visit duration, and returning visitor rates.

Technological Trends: Insight into search engine market share (Google, Bing, etc.) and browser usage (Safari, Chrome, Firefox, IE) across different years and territories.

3. Human Resources & Operational Scaling
Workforce Distribution: Analysis of Full-Time Equivalent (FTE) staff counts by state.

Regional Rankings: Identification of the most labour-intensive regions and their correlation with revenue output.

Technical Skills Demonstrated
Data Modelling: Created a dedicated 'Calculations' layer to aggregate raw data into summary tables using advanced Excel functions.

Trend Analysis: Built dynamic trend indicators to visualise growth patterns over a four-year period.

Dashboard Design: Focused on a clean, professional layout that allows stakeholders to filter data by Country and Year.

Reporting: Implemented ranking systems to highlight top-performing states by revenue and efficiency.

Repository Structure
MY Compnay.xlsx: The primary Excel workbook containing the live dashboard, calculations, and raw data.

Data/: Exported CSV files representing the data architecture:

FinanceData: State-level financial and staffing metrics.

SocialData: Digital marketing and web traffic statistics.

Calculations: The logic layer used to drive dashboard visualisations.

Data Source
Please note that all data used in this project is fictional and was generated for demonstration purposes. It is intended to showcase data visualisation and analytical capabilities rather than represent actual corporate figures.

How to Use
Download or clone the repository.

Open MY Compnay.xlsx in Microsoft Excel (ensure macros/links are enabled if prompted).

Navigate to the Dashboard tab to interact with the visualisations using the provided slicers or filters.
